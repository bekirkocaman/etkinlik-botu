# Güvenlik

## Hassas dosyalar

- `.env` — SerpAPI anahtarı
- `kimlik.json` — Google service account

Bunları issue veya PR’da paylaşmayın.

## Anahtar sızıntısı

Public repoda anahtar göründüyse [SerpAPI](https://serpapi.com/manage-api-key) panelinden **hemen yenileyin**.

Güvenlik sorunu için public issue açmayın; GitHub Security Advisory kullanın.
